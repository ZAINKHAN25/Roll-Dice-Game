Welcome to the Roll Dice Game

The Dice Game is a simple two-player JavaScript web application where players take turns rolling a six-sided die, accumulating points with each roll. The game features an automatic switch of turns when a player rolls a 1, and the first player to reach a specified winning score emerges victorious. With the ability to restart the match, it provides an engaging and interactive gaming experience. Feel free to clone the repository, customize the winning score, and enjoy a friendly competition!
